Features
Dashboard
Your command centre
See at a glance how many quotes, invoices, and customers you have, plus your total outstanding balance. Recent documents listed for quick access.
Quotes
Create & manage quotes
Sequential reference numbers (e.g. QT-0001). Set a valid-until date, add line items, and write notes. Status: Pending, Accepted, Declined.
TipOnce accepted, recreate as an invoice â€” open a new invoice and copy the details across.
Invoices
Bill your customers
Includes GST auto-calculation, due dates, and status tracking (Unpaid, Paid, Partial). Paid invoices display a PAID stamp on printed output.
Print / PDF
A4 print-ready output
Click Preview & Print on any saved document. Use your browser's print dialog to save as PDF. Set margins to "None" for the cleanest result.
Data Storage
100% local & private
All data saves in your browser's localStorage. Nothing is sent to any server. Data persists between sessions. Avoid clearing site data to preserve records.
Line Items
Flexible line entry
Each line has description, optional note, quantity, and rate. Totals calculate instantly. Use notes for part numbers, model refs, or labour codes.
TipNotes appear subtly beneath the description on the printed document.
FAQ
Can I use this offline?
Yes â€” once loaded in your browser, Quotient works entirely offline. The only exception is the Google Fonts used for typography; if offline, your browser falls back to system fonts.
Will my data be lost if I close the browser?
No. All data is saved automatically to your browser's localStorage whenever you click Save. It persists across browser restarts. However, clearing site data or using Incognito mode will not retain data.
How do I set GST to 0%?
Go to Settings â†’ Invoice & Quote Settings and set the GST Rate field to 0. Save Settings. All new documents will show $0.00 GST.
Can I edit a quote or invoice after saving?
Yes â€” click any document row in the list or on the Dashboard. It opens in the Create/Edit tab pre-filled with all data. Make your changes and click Save to update.
How do I delete data and start fresh?
Each document has a delete button (trash icon) visible on hover. To clear everything, open browser DevTools â†’ Application â†’ Local Storage and remove the "quotient-data" key, then refresh.
